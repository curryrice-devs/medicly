## 🎯 Overview

Medicly revolutionizes physical therapy through AI-powered movement analysis, combining computer vision with large language models to provide real-time feedback on patient exercises. Our platform bridges the gap between clinical expertise and accessible rehabilitation, making quality physical therapy guidance available anywhere.

## ✨ Key Features

- **🤖 AI Movement Analysis**: Real-time pose detection with intelligent feedback using MediaPipe and Claude AI
- **📹 Video Processing**: Upload and analyze patient exercise videos with detailed movement breakdowns
- **🔬 3D Anatomy Visualization**: Interactive 3D models using BioDigital Human API for patient education
- **👩‍⚕️ Patient Management**: Comprehensive session tracking and progress monitoring
- **⚡ Real-time Streaming**: Live video analysis with instant pose landmark detection
- **📊 Smart Recommendations**: AI-generated exercise suggestions based on patient conditions
- **🔒 Secure Storage**: HIPAA-compliant data handling with Supabase integration
